15 Second Countdown Timer

purpose: To create a 15 second coundown timer which had a visual output display on an 8x8 LED matrix. The audio output was a speaker which sounded off at 3 different frequencies when the timer reached 0.

hardware used: 12 bit ripple counter, 4 bit counter, EPROM, 4x1 Multiplexer, 3x8 decoder, LM386 Audio Amplfier

lab tools used: oscilloscope, multimeter, DC power source, Digital Laboratory, function generator

A video of the hardware is posted to the right, for more info please visit my Admissions Blog.
